# Zomato Data Analysis (Python)

Simple analysis of Zomato dataset using Pandas, NumPy, Matplotlib, and Seaborn.

## What it does

* Cleans data
* Finds top restaurants (by votes)
* Analyzes restaurant types & online ordering
* Visualizes data (bar charts, heatmaps)

## Tools

* Python
* Pandas, NumPy
* Matplotlib, Seaborn

## Run

Upload dataset → run notebook in Jupyter/Colab

## Output

Basic insights + visualizations for decision-making
